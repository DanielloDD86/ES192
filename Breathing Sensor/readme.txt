The following sensor is very easy to set up and use:
- configure arduino secrets to the available/desired wifi network
- compile and upload code to arduino
- unplug computer and lay the sensor down on workbench and let rest for 60s (this is so the thread has time to return to neutral shape before auto calibration runs)
- turn battery pack on
- if sensor has been idle for more than 3h click the reset button on the arduino before waiting for website
- wait for website to turn on and look at the shown voltage, remember it
- put the breathing sensor on around the abdomen and clip in the buckle
- adjust thightness or position until voltage when breathing in is shown on screen as "the original voltage-0.2V"
- once calibrated to that point, the sensor should operate normally and record breaths
